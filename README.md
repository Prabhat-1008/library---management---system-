# Library Management System (Java + MySQL)

This is a simple console-based Library Management System project made using Java and MySQL for academic purpose.

## Features

- Admin Login
- Add Book
- View Books
- Add Member
- Issue Book
- Return Book
- Fine Calculation (₹5 per day after 7 days)

## Technologies Used

- Java
- JDBC
- MySQL
- Maven

## How to Run

1. Create database using provided SQL script.
2. Update MySQL username and password in DBConnection.java.
3. Run Main.java file.

## Fine Logic

Books can be kept for 7 days without fine.
After 7 days, ₹5 per extra day is charged.
